Project Details

This project is for predicting insurance costs. Users input their age, gender, BMI, number of children, region, smoking status, and health condition. The app processes the data, encodes the region information, and uses pre-trained machine learning models (Linear Regression, Decision Tree, and Random Forest) to make predictions. These predictions are stored in an SQLite database and displayed on the results page. The app provides a user-friendly interface for estimating insurance costs using multiple prediction models.
